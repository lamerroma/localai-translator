# LocalAI Перекладач

> **Зміна підходу (2026-05-29)**
> Беремо за основу проєкт [local-docx-translator](https://github.com/petkovplamen1989/local-docx-translator).
> Ключові ідеї, які запозичуємо: збереження форматування через роботу на рівні XML-структури DOCX,
> «Smart Layout Protection» (автозменшення шрифту при збільшенні тексту), пакетна обробка документів.
> Адаптуємо під Ollama та веб-інтерфейс на FastAPI.
>
> Перехід з LocalAI на **Ollama** зумовлений наявністю відеокарти з **12 GB VRAM** —
> TranslateGemma 12B повністю вміщається у VRAM та працює значно швидше порівняно з CPU-інференсом.



Локальний веб-сервер для перекладу тексту та документів через LLM.  
Розроблений для внутрішнього офісного використання без виходу в інтернет.

---

## Можливості

- Переклад тексту з потоковим виводом токенів у реальному часі
- Переклад файлів: **PDF**, **DOCX**, **TXT**
- 34 мови, автовизначення мови джерела
- Налаштування через веб-інтерфейс без перезапуску сервера
- Зупинка перекладу в будь-який момент
- Завантаження перекладеного файлу одним кліком

---

## Стек

| Компонент   | Деталі |
|-------------|--------|
| Сервер      | FastAPI + Uvicorn |
| LLM бекенд  | Ollama (OpenAI-сумісний API) |
| Модель      | TranslateGemma 12B |
| Адреса LLM  | `http://localhost:11434/v1` |
| Порт сервера | `7860` |

---

## Запуск

```bash
pip install fastapi uvicorn requests pypdf python-docx
ollama pull translategemma:12b
python translate_server.py
```

Інтерфейс: `http://<ip-сервера>:7860`
